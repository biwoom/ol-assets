## 독서뷰 

- [x] 문서뷰 view모드 페지 : 문서뷰의 view모드 관련 기능, 스타일 모두 제거. view모드 아이콘 토글은 독서뷰로 연결되게 변경. 문서뷰는 편집모드 편집시에만 사용.
- [x] 독서뷰 생성 : 독서뷰 새롭게 생성. 스타일은 홈 렌딩 스타일 적용. 
- [x] 독서뷰 헤더 스타일 : 홈 렌딩뷰 헤더 스타일과 메뉴명 동일하게 적용, 다만 스크롤 다운 하면 사라지고 업할 때 다시 나타나게 조정.
- [x] 독서뷰 사이드바 목록 : 독서뷰 사이드바에는 문서뷰 사이드바의 문서목록만 출력하고 about, 휴지통은 표시하지 않는다. 사이드바 박스는 배경 투명하고 조정하거나 독서뷰 배경과 같은 컬러 적용. 문서목록만 텍스트만 부드럽게 나타나고 부드럽게 사라지게 적용. 기본적으로 숨김상태. 햄버거 아이콘 클릭시 나타나고 한 번 더 클릭시 사라짐. 좌측화면 가장 끝에서 나타나는 것이 아닌 중앙 문서 본문영역 바로 좌측 옆에 나타나게 위치 조정.
- [x] 독서뷰 목차 : 독서뷰 사이드바와 같이 배경 투명하고 텍스트만 부드럽게 나타나고 부드럽게 사라지게 적용. 중앙 문서 본문영역 바로 우측 옆에 나타나게 위치 조정.
- [x] 독서뷰 사이드바와 목차 토글 아이콘 : 햄버거 아이콘 혹은 원형 링 형태의 아이콘 중 선택. 아이콘 hover 시 사이드바와 목차 텍스트 나타나고 hover 벗어나면 사이드바와 목차 텍스트 사라지게. 아이콘 클릭하면 계속 나타나고 본문 스크롤해도 sticky 스타일로 고정되고 아이콘 재클릭하면 다시 사라제게 조정.
- [x] 독서뷰 하단 페이지 이동 버튼 : 독서뷰 중앙본문 하단영역은 문서뷰 .dv-foot 기능 적용하여 다음페이지, 이전 페이지 이동할 수 있게 한다.
- [x] 독서뷰 글자크기 조정 버튼 : 문서뷰 .dv-meta 영역 가장 우측에 +, - 아이콘 추가하여 본문 텍스트 글자를 사용자에 맞게 조정할 수 있게 한다.
