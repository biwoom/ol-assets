# OL BOOK 워크플로우 초안

[기본 용어]
- Cut(장면) : 글(Text) : 카드 내용 한 문장에 대응 : 한 문장(수메다는 깊은 생각에 잠겼다)

- Scene(사건) : 문단(Paragraph) : 카드 내용 한 단락에 대응 : 한 사건(수메다가 연등불의 만나 수기를 받는 사건)

- Sequence(단막) : 문서(Document) : 칸반 카드에 대응 : 이 땅에 오시기까지(수메다의 발원 이야기)

- Episode(회차/편) : 장 (Chapter) : 칸반 칼럼에 대응 : 제1장 탄생과 성장

- Season(작품) : 책(Book) : 칸반 전체에 대응 : 부처님 생애

---

## 한 Sequence(단막) : 문서(Document) 제작 워크플로우 초안

[인포레이어]
원본 텍스트 수집
텍스트 간 상호 비교 
텍스트 문서 선정 - 페이지(page) 완성

[서사레이어]
선정된 문서 대본화
대본 콘티화 작업
콘티 영상화 기획 - 전체 스타일 철학과 전략 구상 / 캐릭터 구성 / 배경 구성 / 소품 구성
콘티 장면별 이미지 제작 - AI 서비스 이용: 미드저니, chatGPT 이미지, 클로드 디자인 등

[인포레이어]
문서(Document)에 장면이미지 삽화로 추가

[서사레이어]
장면이미지에서 캐릭터, 배경, 소품 분리.
캐릭터, 배경, 소품을 SVG 로 변경. 가능한 부분만. 

콘티에 맞게 이미지 요소들 Cut(장면)별 배치.
Cut(장면)으로 Scene(사건) 구성.
Scene(사건)으로 Sequence(단막) 구성

Cut(장면)별로 요소들 애니메이션 세부 조정
Cut, Scene 전환시 애니메이션 제작과 세부 조정

[인포레이어] + [서사레이어]
두 레이어 정보 연결, 각 레이어 UI 간 연결, UI표준안 마련
