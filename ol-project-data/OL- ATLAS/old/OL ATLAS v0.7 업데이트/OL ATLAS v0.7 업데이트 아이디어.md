## OL 듀얼 런타임 설계 아이디어

- [ ] 기존 v0.7은 편집모드와 리딩모드 통합버전. 이 통합버전에서 카드편집, 문서편집 기능, 휴지통뷰, 리스트뷰 등의 사용자에게 불필한 편집기능을 감추고 리딩기능만 담아 새롭게 저장하게 하는 기능 탑재. 이 리딩모드 버전은 OL BOOK 배포버전이 되는 방식. 제작자는 통합버전에서 컨텐츠 제작한 후 배포시 배포버전 저장버튼을 통해 리딩버전만 담은 OL BOOK.html 생성하여 배포하는 구조.
- [ ] 듀얼 런타임 설계 :  ATLAS / BOOK 책임 분리,  `ui/author/` vs `ui/reader/` vs `ui/shared/` 디렉토리 재배치, Dual Bundle 구조 (두 번들이 한 dist에 공존), 런타임 export 함수 (`exportBook()`), Reader Manifest 스펙, Export Manifest 스펙 (어떤 데이터를 BOOK에 넣을지)
- [ ] OL BOOK에 커버페이지 만들 수 있는 입력폼 만들기.
- [ ] 제거뷰: 리스트뷰, 휴지통, 카드뷰, 홈렌딩(커버페이지 제작뷰로 전환)
- [ ] 칸반보드 수정 >> 카드보드.카드보드는 문서뷰 링크 목차 역할, 태그 필터링 결과 페이지.
- [ ] 제거기능 : 카드모달, 내용편집기능, 저장과 열기 버튼, 칸반보드 카드추가-칼럼추가 버튼, 칸반보드 칸반-카드 전환토글, 
- [ ] 유지: 사이드바 칼럼, 그룹, 태그 섹션, ABOUT
- [ ] 사이드바는 기본 감춰진 상태, 왼쪽으로 커서 이동시, 로고 호버시에 화면에 뜬 구름스타일로 나타나게 스타일 변경.
- [ ] 최상단 메인 메뉴 영역은 가장 오른쪽 OL로고, 가장 왼쪽 검색폼, 다크모드 토글. 문서뷰에서는 이 영역도 기본값으로 감춰진 상태로 변경. 커서가 상단으로 이동시 나타남.
- [ ] 브라우저 새로고침시 업데이트한 내용 유실 경고창 만들기, 카드 삭제 시에도 경고창 만들기. 시스템 경고창이 아닌 div 태그 통한 경고창 만들기.

## v0.7 새로운 UI 아이디어

1. 사이드바는 기본 숨김 상태. 로고 옆 햄버거 아이콘 클릭시 나타남.
2. 사이드바 칼럼과 그룹 영역 제거. 태그 영역 제거. 태그 검색은 탭네비게이션 상단 검색바로 통합

4. 사이드바 ABOUT, 휴지통 보존
5. 로고 클릭시 홈렌딩 페이지 제거. 로고 클릭시 카드보드로 이동.
