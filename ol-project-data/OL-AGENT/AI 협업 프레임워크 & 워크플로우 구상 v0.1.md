# AI 협업 프레임워크 & 워크플로우 구상 v0.1
## OL-CREW 

- OL번역원 개념 폐기. OL-CREW 개념 도입.
- OL-CREW : 오케스트레이션 AI 에이전트와 협업하여 불교콘텐츠를 창작하는 프로젝트. OL 프로젝트 소속이며 철학계승. 
- OL-CREW 세부 프로젝트 : OL-DESK, OL-WRITER, OL-TOON, OL-STUDIO. 
	- 각 크루 세부 프로젝트 구성: 1) Astro 웹빌더 기반 UX. 2) Contents-Asset. 3) AI-Agent 설정 지침. 이 세 개 층으로 구성.
	- 'Astro 웹빌더 기반 UX' - 인간 감수자의 수정, 확정, 최종 창작물 완료가 이루어지기 때문. 
	- 'AI-Agent'는 AI 오케스트레이션 체계. 전문 콘텐츠 제작에게 외주 요청과 유사. 
	- 'Contents-Asset'은 AI와 협으로 생산된 순수 창작 자산과 유사.
	- OL-DESK : OL-DESK-CREW. 불고전문헌 번역 중심 불교 콘텐츠 창작크루.
	- OL-WRITER : OL-WRITER-CREW. 텍스트 중심 불교 콘텐츠 창작 크루.
	- OL-TOON : OL-TOON-CREW. 이미지 중심 불교 콘텐츠 창작 크루.
	- OL-STUDIO : OL-STUDIO-CREW. 영상 중심 불교 콘텐츠 창작 크루.
- AI 오케스트레이션과 협업하지만 페이퍼클립과 같은 특정 프로젝트에 한정하여 개발하지 않는다. 에이전트 보편성을 유지하면서 설정지침들을 제작한다.
- 지금은  OL-WRITER, OL-TOON, OL-STUDIO 개발하지 않음. OL-DESK 에 집중.
- OL-DESK 프로젝트 기반이 안정되면 이 구조를 응요하여  OL-WRITER, OL-TOON, OL-STUDIO 순차적 개발 시작.

## AI협어 워크플로우 종류와 상태정보 변경

1. 1-2차번역 워크플로우 : A-관리매니저 주도 
2. 2차 휴먼 워크플로우 : OL-DESK 주도
3. 3차번역 워크플로우 : B-확정매니저 주도 
4. 3차 번역본 및 4차원고 수정 워크플로우 : 4차 원고로 완료된 번역본이라 할지라도 OL-DESK에서 본문, 용어, 각주, 태그 수정시 다시 3차가번역본이 되어 3차번역 워크플로우 진행. 원고 상태정보는 '3차번역 워크플로우 진행중'으로 변경됨.

더 정교하게 세분한 버전: 완료본을 수정한다고 해서 단순히 “과거 상태로 되돌아간다”기보다, **새 revision cycle이 열린다**고 보는 것이 좋음. 더 새분하되 DESK UX에 표시되는 것은 한글표기로 한다.
```
완료본 수정 발생
→ revision_needed
→ draft3_preliminary_reopened
→ draft3_requested
→ draft3_generated
→ final_review
→ final_done
```

## OL-DESK 레이아웃

대시보드 스타일. 참조사이트: https://ui.shadcn.com/create 
메인 메뉴와 페이지 : 각 페이지가 상단 헤더 메뉴로 구성. 중앙정렬.
- AGENET : 개별 agent 활동평가. 사이드바에 개별 에이전트 목록. 본문에 전체 에이전트 활동기록. 사이드바 개별 에이전트 클릭이벤트 > 해당 에이전트 활동 로그와 감수자 평가 기록 창.
- 문헌 : 원본문헌 리스트 페이지. 원본 문헌들의 워크플로우 진행상태 리스트. 사이드바 준비중, 진행중, 감수필요, 완료됨 상태 필터 버튼. 개별 문헌 클릭이벤트 > 해당 문헌 번역페이지로 이동.
- 번역 : 번역본 확인 및 수정 페이지. 1, 2, 3, 4차 번역본의 본문, 용어, 각주, 태그 수정. 실제 수정은 3, 4본만 가능. 번역본 별 탭 페이지.
- 용어 : 개별 프로젝트 전체 누적 용어 및 신규용어 페이지, 개별 용어 추가, 수정, 삭제 기능.
- 각주 : 개별 프로젝트 전체 누적 각주 및 신규각주 페이지, 개별 각주 추가, 수정, 삭제 기능.
- 태그 : 개별 프로젝트 전체 누적 태그 및 신규각주 페이지, 개별 태그 추가, 수정, 삭제 기능.
- 관리 : 개별 agent 활동평가 정보와 OL-DESK 코드 업데이트 아이디어 담은 관리문서 생성, 이전 관리문서 저장소. 깃헙 업로드 기능. 이슈별로 리스트로 그룹화. 이슈별로 평가와 완료 체크 기능.
