### 세션 내부에서 모드 전환 방법

터미널 세션이 이미 켜져 있는 상태라면 명령어를 다시 입력할 필요 없이 아래 단계를 따릅니다.

- 터미널 창에서 **`Shift + Tab`** 키를 누릅니다.    
- 권한 모드가 순환(`default` ➡️ `acceptEdits` ➡️ `plan` ➡️ `auto`)되는데, `auto`가 표시될 때 멈추시면 됩니다.
  
```
 실행 시 참고사항 (작동이 안 될 때)
- **버전 및 모델 요구사항:** Claude Code `v2.1.83` 이상 버전이 필요하며, **Claude 4.6 Sonnet** 또는 **Opus** 모델 환경에서만 지원됩니다. (이전 4.5 이하 버전 모델은 지원하지 않습니다.)
- **계정 권한 (Team / Enterprise):** 만약 조직 계정을 사용 중이라면, 팀 관리자(Admin)가 Claude Code 관리자 설정(Organization Settings -> Claude Code)에서 Auto mode를 '허용(Enable)' 상태로 켜두어야 개발자가 CLI에서 이 명령어를 사용할 수 있습니다.
```
