## 1. 브랜드 개요

### 1.1 브랜드명
- **한글**: 올
- **영문**: OL
- **발음**: [올] / [ol]

### 1.2 핵심 정의
올은 방대한 불교 논서와 경전을 디지털 환경에 최적화된 **'단일 주제 깊이 파기(Deep Dive)' 인터랙티브 웹북**으로 재구축하는 불교 학습 플랫폼입니다. 백과사전식 나열을 지양하고, 문맥과 개념이 유기적으로 연결된 살아있는 학습 도구를 지향합니다. DharmaBase OL은 **'개인 지식 베이스(PKB)'** 와 **'인터랙티브 웹북'** 이 결합된 새로운 형태의 수행(Learning) 플랫폼입니다. DharmaBase:개인 지식 베이스, OL:인터랙티브 웹북에 해당합니다. 이 브랜드의 핵심이자 특별함은 OL(인터랙티브 웹북)에 있습니다.

### 1.3 태그라인

#### 한국어
- **주**: 실처럼 이어진 지혜
- **부**: 모든 지혜가 이어지는 곳
- **부**: 하나하나 따라가는 깨달음

#### 영어
- **주**: Thread by thread, truth unfolds
- **부**: Where all wisdom weaves together
- **부**: The fabric of understanding

#### 한영 혼합
- **올 · 올 따라 가는 길** (Following the thread)

### 1.4 슬로건
**"지혜의 올을 엮다"** (Weaving the Wisdom)

## 2. 브랜드 철학

### 2.1 핵심 가치

#### A. 연결 (Connection)
올 하나하나가 모여 천을 이루듯, 고립된 텍스트들을 살아있는 연결망으로 엮습니다. 모든 개념, 경전, 논서는 독립적이면서도 서로 연결되어 있습니다.

#### B. 깊이 (Depth)
천의 표면을 보는 것이 아니라, 올 하나하나를 따라가며 깊이 탐구합니다. 백과사전식 나열이 아닌, 단일 주제에 대한 심층적 이해를 추구합니다.

#### C. 여정 (Journey)
각자가 자신의 올을 선택해 따라가는 학습 여정입니다. 같은 천이지만 각자 다른 경로로, 사용자가 직접 자신만의 천을 짜는 경험을 제공합니다.

### 2.2 디자인 철학

#### 최소주의 (Minimalism)
- 한 음절, 두 글자의 절제된 아름다움
- 복잡함을 덜어내고 본질에 집중
- "덜어낼수록 더 많이 담는다"는 동양 미학

#### 유기성 (Organicity)
- 정적 데이터베이스가 아닌 살아 숨쉬는 연결망
- 사용자의 학습에 따라 성장하고 변화하는 구조
- 자연스러운 흐름과 연결

#### 개방성 (Openness)
- 로그인 장벽 없이 즉시 접근 가능
- 지식은 모두에게 열려있어야 한다는 신념
- 사용자 데이터의 완전한 통제권 보장

## 3. 의미 구조 분석

### 3.1 한국어 층위

#### 층위 1: 올 (실의 올, 직물의 세로줄)
**표면 의미**
- 섬세함, 디테일, 텍스트의 기본 단위
- 한 올 한 올 정성스럽게 엮어진 것

**불교적 연상**
- **수트라(Sutra = 실)**: 경전의 어원 자체가 '실'입니다. 산스크리트어 'sūtra'는 실, 줄을 의미하며, 부처의 말씀을 실처럼 꿰어 놓았다는 뜻입니다.
- **인연의 실**: 모든 존재가 인연의 실로 연결되어 있다는 연기법(緣起法)
- **업(業)의 직조**: 선업과 악업의 올이 모여 현생의 천을 이룬다는 업보 사상

**프로젝트 연결**
- Neural Network의 경위선(經緯線) 같은 연결망
- 각 문장, 각 개념이 하나의 올
- 올들이 교차하며 만드는 지식의 직물

#### 층위 2: 오르다 (상승, 향상)
**표면 의미**
- 위로 향함, 발전, 성장

**불교적 연상**
- **상구보리(上求菩提)**: 위로 깨달음을 구한다
- **하화중생(下化衆生)**: 아래로 중생을 교화한다

**프로젝트 연결**
- Deep Dive의 역설: 깊이 파고들수록 높이 올라감
- 학습을 통한 의식의 상승
- 무명(無明)에서 명(明)으로의 상승

#### 층위 3: 올 (모두, 전부)
**표면 의미**
- 올해, 올것처럼 '모든' 것을 포괄

**불교적 연상**
- **일체(一切)**: 모든 것
- **법계(法界)**: 모든 현상과 진리의 총체
- **전법륜(轉法輪)**: 모든 존재에게 진리를 전함

**프로젝트 연결**
- 포괄적 지식 체계
- 경전과 논서의 전체를 아우름
- 부분이면서 동시에 전체

### 3.2 영어 연상

#### All (모든 것)
- 완전성, 총체성
- "All connected" - 모든 것이 연결됨
- "All in one" - 하나 안에 모든 것

#### OL' (old의 구어체)
- 오래된, 고전적
- 고대의 지혜에 대한 현대적 접근
- 전통과 혁신의 조화

### 3.3 불교적 상징의 심화

#### 직조(織造)의 은유
```
날실(經) = 경전(經典)
씨실(緯) = 논서(論書)
올 하나하나 = 개별 개념/문장/단어
천 전체 = 법계(法界)의 총체
직조 행위 = 학습과 깨달음의 과정
```

#### 인드라망(因陀羅網)과의 일치
화엄경의 핵심 이미지인 **인드라의 그물**과 정확히 일치:
- 각 올이 다른 모든 올과 연결
- 하나를 당기면 전체가 움직임 (연기법)
- 부분과 전체가 상즉상입(相卽相入)
- 각 보석(올)이 다른 모든 보석을 비춤 (상호조명)

#### 중도(中道)의 실현
- 올 하나하나는 독립적이지도, 완전히 융합되지도 않음
- 데이터베이스 설계 원칙인 **원자성(Atomicity)**과 **관계성(Relationship)**의 불교적 해석
- 유(有)도 아니고 무(無)도 아닌 중도의 길

## 4. 프로젝트 정체성과의 정합성

### 4.1 "살아있는 연결망"과의 일치

| 개념 | 올의 은유 | 기술적 구현 |
|------|----------|------------|
| 노드 (Node) | 올 하나하나 | 개별 문서, 개념, 문장 |
| 연결 (Connection) | 올이 만나는 점 | 하이퍼링크, 관계 메타데이터 |
| 네트워크 (Network) | 천 전체 | 전체 지식 그래프 |
| 학습 여정 | 직조 행위 | 사용자의 탐색 경로 |

**실현 방식**
- 모든 문서는 고립되지 않고 상호 연결됨
- Frontmatter 메타데이터 기반 빌드 시점 관계망 생성
- 사이드바/인라인 주석을 통한 즉시 이동
- 사용자는 올을 따라가듯 개념을 탐색

### 4.2 "Deep Dive"와의 일치

| 학습 단계 | 올의 은유 | 사용자 경험 |
|----------|----------|------------|
| 표면 이해 | 천의 표면 보기 | 목차, 개요 파악 |
| 깊이 탐구 | 올 하나 따라가기 | 단일 주제 집중 학습 |
| 통찰 획득 | 올의 교차점 발견 | 개념 간 연결 이해 |
| 전체 조망 | 천의 짜임 파악 | 지식 구조 통찰 |

**차별화**
- 백과사전식 나열 지양
- 한 번에 하나의 올(주제)에 집중
- 점진적으로 깊어지는 학습 경험
- 표면에서 본질로의 여정

### 4.3 "로그인 없는 개인화"와의 일치

**철학적 정합성**
- 각자가 자신의 올을 선택해 따라감
- 같은 천이지만 각자 다른 경로
- 지식은 공유되지만 여정은 개인적

**기술적 구현**
- LocalStorage 기반 개인 데이터 저장
- JSON Export/Import 백업 기능
- 형광펜, 메모, 북마크의 로컬 관리
- 프라이버시 완전 보장

## 5. 브랜드 네이밍 전략

### 5.1 기본 원칙: "올" 단독 사용

#### 공식 명칭
```
브랜드명: 올 (OL)
제품 설명: 인터랙티브 불교 웹북 플랫폼
영문: OL · Interactive Buddhist Webbooks
```

#### 핵심 이유
1. **완결성**: "올" 자체가 완결된 브랜드
2. **단순성**: Deep Dive는 방법론이지 제품명이 아님
3. **확장성**: 향후 어떤 제품도 "올" 아래 자연스럽게 배치 가능
4. **강력함**: Google, Apple처럼 브랜드만으로 충분

### 5.2 사용 가이드라인

#### ✅ 자연스러운 표현
```
"올에서 화엄경을 읽어봤어?"
"올로 공부하면 개념들이 연결돼서 이해가 쉬워"
"올은 불교 경전과 논서를 인터랙티브하게 탐구하는 플랫폼입니다"
"Have you tried studying on OL?"
"Explore Buddhist wisdom on OL"
```

#### ❌ 피해야 할 표현
```
"올북에서..." (어색, 올의 순수성 희석)
"올다이브로..." (의미 불명, 조합 억지)
"웹북올..." (어순 부자연)
"딥다이브올..." (불필요한 중복)
```

#### 필요시 활용 (일반명사로서)
```
"올 웹북" - 제품 카테고리 설명 시
"올의 인터랙티브 기능" - 기능 설명 시
"올 플랫폼" - 전체 생태계 지칭 시
```

### 5.3 도메인 전략

#### 1순위
- **ol.io** - 글로벌, 기술 친화적, 간결함
- **올.kr** - 한국, 직관적, 로컬 정체성

#### 2순위 (보완)
- **ol.study** - 학습 플랫폼 정체성 명확화
- **studyol.com** - SEO 대체 도메인

### 5.4 향후 확장 시 제품 계층 구조

```
올 (OL) - 메인 플랫폼 브랜드
│
├─ 웹북 - 제품 카테고리 (별도 이름 불필요)
│  ├─ 화엄경
│  ├─ 금강경
│  └─ 중론
│
├─ 올길 (OL Paths) - 학습 경로 기능 (Phase 2+)
│  ├─ 초심자 과정
│  └─ 주제별 트랙
│
├─ 올노트 (OL Notes) - 개인 학습 도구 (Phase 2+)
│
└─ 올모임 (OL Circles) - 커뮤니티 (Phase 3+)
```

**원칙**
- Phase 1에서는 "올"만 사용
- 제품 다양화 시점에 서브 브랜드 도입
- 모든 서브 브랜드는 "올"을 접두어로 유지

## 6. 브랜드 스토리텔링

### 6.1 브랜드 스토리 구조

#### Hero Message (핵심 메시지)
```
지혜의 올을 엮다
(Weaving the Wisdom)

흩어진 불교의 경전과 논서를
연결된 하나의 맥락으로
```

#### Value Proposition (가치 제안)

**1. 한 올 (Deep Dive)**
단 하나의 주제를 끝까지 파고듭니다.
백과사전처럼 모든 것을 나열하지 않습니다.
당신이 선택한 올을 따라, 깊이 들어갑니다.

**2. 짜임 (Context)**
텍스트와 텍스트 사이의 숨겨진 관계를 봅니다.
고립된 문장이 아니라, 연결된 맥락을 읽습니다.
올이 교차하는 지점에서 통찰이 일어납니다.

**3. 전체 (Wholeness)**
마침내 드러나는 거대한 지혜의 지도를 경험합니다.
부분을 이해하면 전체가 보이고,
전체를 보면 부분이 빛납니다.

### 6.2 "왜 올인가" 설명 콘텐츠

#### 런칭 페이지용 스토리

**도입**
```
올 하나는 작습니다.
하지만 수천 개의 올이 모이면 천이 됩니다.

경전(經典)의 '經'은 날실을 뜻합니다.
부처의 말씀을 실처럼 꿰어 놓았다는 뜻입니다.

올은 그 경전을 다시 풀어,
당신이 직접 짜볼 수 있게 합니다.
```

**전개**
```
화엄경은 인드라의 그물을 이야기합니다.
무한한 보석들이 그물처럼 연결되어,
각각이 다른 모든 것을 비춥니다.

올은 그 그물의 디지털 버전입니다.
모든 개념, 모든 문장이 서로 연결되어
하나를 읽으면 전체가 움직입니다.
```

**결론**
```
올에서는 길을 잃지 않습니다.
당신이 선택한 올이 길이 됩니다.

깊이 파고들수록 높이 올라가고,
부분을 이해할수록 전체가 보입니다.

지혜의 올을 따라,
당신만의 천을 짜보세요.
```

### 6.3 브랜드 보이스 (Brand Voice)

#### 톤 (Tone)
- **명상적**: 서두르지 않고 깊이 생각하게 함
- **초대적**: 강요하지 않고 탐험을 권유함
- **지혜로운**: 가르치려 하지 않고 통찰을 나눔

#### 언어 스타일
- **간결함**: 짧은 문장, 명확한 의미
- **시적임**: 은유와 비유의 자연스러운 사용
- **존중함**: 사용자를 학습자이자 탐험가로 대우

#### 피해야 할 톤
- 교조적, 권위적
- 과도하게 기술적
- 상업적, 마케팅적

## 7. 콘텐츠 전략 및 적용

### 7.1 웹북 콘텐츠 구조화 원칙

#### A. "올" 개념의 콘텐츠 적용

**개별 올 단위 정의**
- 문장 레벨: 각 문장이 하나의 올
- 개념 레벨: 각 불교 용어/개념이 하나의 올
- 섹션 레벨: 각 주제가 하나의 올

**올의 교차점 (연결점)**
```markdown
## Frontmatter 예시
```

## 2. 상세 분석

### 2.1 의미론적 분석 (Semantics)

브랜드 정의서에 명시된 철학인 **"부분과 전체의 조화(Indra's Net)"** 관점에서 해석할 때 매우 흥미로운 대립과 조화를 이룹니다.

1.  **Ol (전체/All) + Bit (조각/Part)**
    * **철학적 해석:** 'Ol'은 전체(Wholeness)를, 'Bit'은 정보의 최소 단위(Atomicity)를 상징합니다. 이는 화엄경의 **"일미진중함시방(一微塵中含十方, 티끌 하나 속에 우주가 담겨 있다)"** 사상과 절묘하게 맞아떨어집니다.
    * **기술적 해석:** 브랜드 정의서 4.3에서 언급한 **"원자성(Atomicity)"**과 **"관계성"** 중, 'Bit'은 원자성을 강력하게 대변합니다.

2.  **디지털 정체성 강화**
    * 기존의 `ol.io`나 `ol.study`가 '행위'나 '상태'에 집중했다면, `olbit`은 이 플랫폼이 **"디지털 데이터"**로 이루어진 공간임을 명확히 합니다. '인터랙티브 웹북'이라는 기술적 속성을 잘 드러냅니다.

### 2.2 영어권 뉘앙스 및 발음 (English Nuance & Pronunciation)

영어권 사용자 입장에서 `olbit`을 접했을 때의 느낌은 다음과 같습니다.

* **발음:** `[ohl-bit]`
    * 발음이 끊어짐 없이 부드럽게 이어지며, 짧고 명쾌합니다.
    * **Orbit(궤도)와의 유사성:** 발음이 'Orbit'과 상당히 유사하게 들립니다. 이는 긍정적인 요소로 작용할 수 있습니다.
        * *연상 이미지:* 지혜의 중심을 도는 궤도, 법륜(Dharma Wheel)의 회전, 순환 등 불교적 메타포와 연결될 수 있습니다.
* **어감 (Vibe):**
    * **Tech-savvy:** 'Bit', 'Qubit', 'Fitbit' 등의 영향으로 현대적이고 기술적인 플랫폼이라는 인상을 줍니다. 고전 불교 텍스트를 다루지만, 그 방식은 매우 현대적일 것이라는 기대감을 줍니다.
    * **가벼움 (Casual):** 'Bit'은 '작은 조각'이라는 뜻이 있어, 방대한 경전을 다루기에 다소 가볍게(Lightweight) 느껴질 수 있습니다. 하지만 이는 "부담 없이 접근 가능한(Minimalism)" 브랜드 철학과는 상통합니다.
* **잠재적 리스크 (Minor Risk):**
    * **"Old Bit":** 빠르게 발음할 경우 "Old bit" (오래된 조각/부스러기)처럼 들릴 수 있습니다. "오래된 지혜(Old Wisdom)"로 긍정적 해석도 가능하지만, "낡은 기술(Legacy tech)"로 오해될 소지는 미미하게나마 존재합니다.

### 2.3 브랜드 정체성(Identity)과의 정합성

브랜드 정의서의 핵심 키워드인 **'실(Thread)'** 및 **'직조(Weaving)'**와의 연결성을 검토합니다.

| 비교 항목 | 브랜드 컨셉 (올/Thread) | 도메인 (olbit) | 정합성 판단 |
| :--- | :--- | :--- | :--- |
| **메타포** | 실, 천, 직조 (아날로그, 연속성) | 비트, 데이터 (디지털, 불연속성) | **상충하지만 보완적** |
| **핵심 가치** | 연결 (Connection) | 단위 (Unit) | **데이터의 '단위'를 강조함** |
| **지향점** | 깊이 파기 (Deep Dive) | 디지털화 (Digitization) | **수단(Tech)을 강조** |

* **Conflict:** 브랜드의 메인 비주얼이나 스토리가 "실과 직조"인데, 도메인 이름이 "비트(Bit)"라면 **아날로그 감성 vs 디지털 네이밍**의 충돌이 발생합니다.
* **Resolution:** 하지만 이 브랜드가 "종이책"이 아닌 **"인터랙티브 웹북"**임을 감안하면, `olbit`은 **"디지털로 짠 실(Digital Thread)"**이라는 새로운 해석을 가능하게 합니다.

### 2.4 TLD (.org)의 적절성

* **신뢰도:** `olbit.org`의 `.org`는 비영리, 지식 공유, 오픈 소스, 커뮤니티의 성격을 강하게 띱니다.
* **철학적 일치:** 정의서의 **"개방성(Openness) - 지식은 모두에게 열려있어야 한다"**는 철학과 `.org`는 완벽하게 일치합니다. 상업적 느낌(.com)이나 지나치게 기술적인 느낌(.io)보다 훨씬 더 브랜드의 진정성을 잘 전달합니다.
