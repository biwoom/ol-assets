**Version 1.0 | 2026.05.18**

## 1. 브랜드 개요

### 1.1 브랜드명
- **한글**: 올
- **영문**: OL
- **발음**: [올] / [ol]

### 1.2 핵심 정의
올은 방대한 불교 논서와 경전을 디지털 환경에 최적화된 **'단일 주제 깊이 파기(Deep Dive)' 인터랙티브 웹북**으로 재구축하는 불교 학습 플랫폼입니다. 백과사전식 나열을 지양하고, 문맥과 개념이 유기적으로 연결된 살아있는 학습 도구를 지향합니다. DharmaBase OL은 **'개인 지식 베이스(PKB)'** 와 **'인터랙티브 웹북'** 이 결합된 새로운 형태의 수행(Learning) 플랫폼입니다. DharmaBase:개인 지식 베이스, OL:인터랙티브 웹북에 해당합니다. 이 브랜드의 핵심이자 특별함은 OL(인터랙티브 웹북)에 있습니다.

### 1.3 태그라인

#### 한국어
- **주**: 실처럼 이어진 지혜
- **부**: 모든 지혜가 이어지는 곳
- **부**: 하나하나 따라가는 깨달음

#### 영어
- **주**: Thread by thread, truth unfolds
- **부**: Where all wisdom weaves together
- **부**: The fabric of understanding

#### 한영 혼합
- **올 · 올 따라 가는 길** (Following the thread)

### 1.4 슬로건
**"지혜의 올을 엮다"** (Weaving the Wisdom)

## 2. 브랜드 철학

### 2.1 핵심 가치

#### A. 연결 (Connection)
올 하나하나가 모여 천을 이루듯, 고립된 텍스트들을 살아있는 연결망으로 엮습니다. 모든 개념, 경전, 논서는 독립적이면서도 서로 연결되어 있습니다.

#### B. 깊이 (Depth)
천의 표면을 보는 것이 아니라, 올 하나하나를 따라가며 깊이 탐구합니다. 백과사전식 나열이 아닌, 단일 주제에 대한 심층적 이해를 추구합니다.

#### C. 여정 (Journey)
각자가 자신의 올을 선택해 따라가는 학습 여정입니다. 같은 천이지만 각자 다른 경로로, 사용자가 직접 자신만의 천을 짜는 경험을 제공합니다.

### 2.2 디자인 철학

#### 최소주의 (Minimalism)
- 한 음절, 두 글자의 절제된 아름다움
- 복잡함을 덜어내고 본질에 집중
- "덜어낼수록 더 많이 담는다"는 동양 미학

#### 유기성 (Organicity)
- 정적 데이터베이스가 아닌 살아 숨쉬는 연결망
- 사용자의 학습에 따라 성장하고 변화하는 구조
- 자연스러운 흐름과 연결

#### 개방성 (Openness)
- 로그인 장벽 없이 즉시 접근 가능
- 지식은 모두에게 열려있어야 한다는 신념
- 사용자 데이터의 완전한 통제권 보장

## 3. 의미 구조 분석

### 3.1 한국어 층위

#### 층위 1: 올 (실의 올, 직물의 세로줄)
**표면 의미**
- 섬세함, 디테일, 텍스트의 기본 단위
- 한 올 한 올 정성스럽게 엮어진 것

**불교적 연상**
- **수트라(Sutra = 실)**: 경전의 어원 자체가 '실'입니다. 산스크리트어 'sūtra'는 실, 줄을 의미하며, 부처의 말씀을 실처럼 꿰어 놓았다는 뜻입니다.
- **인연의 실**: 모든 존재가 인연의 실로 연결되어 있다는 연기법(緣起法)
- **업(業)의 직조**: 선업과 악업의 올이 모여 현생의 천을 이룬다는 업보 사상

**프로젝트 연결**
- Neural Network의 경위선(經緯線) 같은 연결망
- 각 문장, 각 개념이 하나의 올
- 올들이 교차하며 만드는 지식의 직물

#### 층위 2: 오르다 (상승, 향상)
**표면 의미**
- 위로 향함, 발전, 성장

**불교적 연상**
- **상구보리(上求菩提)**: 위로 깨달음을 구한다
- **하화중생(下化衆生)**: 아래로 중생을 교화한다

**프로젝트 연결**
- Deep Dive의 역설: 깊이 파고들수록 높이 올라감
- 학습을 통한 의식의 상승
- 무명(無明)에서 명(明)으로의 상승

#### 층위 3: 올 (모두, 전부)
**표면 의미**
- 올해, 올것처럼 '모든' 것을 포괄

**불교적 연상**
- **일체(一切)**: 모든 것
- **법계(法界)**: 모든 현상과 진리의 총체
- **전법륜(轉法輪)**: 모든 존재에게 진리를 전함

**프로젝트 연결**
- 포괄적 지식 체계
- 경전과 논서의 전체를 아우름
- 부분이면서 동시에 전체

### 3.2 영어 연상

#### All (모든 것)
- 완전성, 총체성
- "All connected" - 모든 것이 연결됨
- "All in one" - 하나 안에 모든 것

#### OL' (old의 구어체)
- 오래된, 고전적
- 고대의 지혜에 대한 현대적 접근
- 전통과 혁신의 조화

### 3.3 불교적 상징의 심화

#### 직조(織造)의 은유
```
날실(經) = 경전(經典)
씨실(緯) = 논서(論書)
올 하나하나 = 개별 개념/문장/단어
천 전체 = 법계(法界)의 총체
직조 행위 = 학습과 깨달음의 과정
```

#### 인드라망(因陀羅網)과의 일치
화엄경의 핵심 이미지인 **인드라의 그물**과 정확히 일치:
- 각 올이 다른 모든 올과 연결
- 하나를 당기면 전체가 움직임 (연기법)
- 부분과 전체가 상즉상입(相卽相入)
- 각 보석(올)이 다른 모든 보석을 비춤 (상호조명)

#### 중도(中道)의 실현
- 올 하나하나는 독립적이지도, 완전히 융합되지도 않음
- 데이터베이스 설계 원칙인 **원자성(Atomicity)**과 **관계성(Relationship)**의 불교적 해석
- 유(有)도 아니고 무(無)도 아닌 중도의 길

## 6. 브랜드 스토리텔링

### 6.1 브랜드 스토리 구조

#### Hero Message (핵심 메시지)
```
지혜의 올을 엮다
(Weaving the Wisdom)

흩어진 불교의 경전과 논서를
연결된 하나의 맥락으로
```

#### Value Proposition (가치 제안)

**1. 한 올 (Deep Dive)**
단 하나의 주제를 끝까지 파고듭니다.
백과사전처럼 모든 것을 나열하지 않습니다.
당신이 선택한 올을 따라, 깊이 들어갑니다.

**2. 짜임 (Context)**
텍스트와 텍스트 사이의 숨겨진 관계를 봅니다.
고립된 문장이 아니라, 연결된 맥락을 읽습니다.
올이 교차하는 지점에서 통찰이 일어납니다.

**3. 전체 (Wholeness)**
마침내 드러나는 거대한 지혜의 지도를 경험합니다.
부분을 이해하면 전체가 보이고,
전체를 보면 부분이 빛납니다.

### 6.2 "왜 올인가" 설명 콘텐츠

#### 런칭 페이지용 스토리

**도입**
```
올 하나는 작습니다.
하지만 수천 개의 올이 모이면 천이 됩니다.

경전(經典)의 '經'은 날실을 뜻합니다.
부처의 말씀을 실처럼 꿰어 놓았다는 뜻입니다.

올은 그 경전을 다시 풀어,
당신이 직접 짜볼 수 있게 합니다.
```

**전개**
```
화엄경은 인드라의 그물을 이야기합니다.
무한한 보석들이 그물처럼 연결되어,
각각이 다른 모든 것을 비춥니다.

올은 그 그물의 디지털 버전입니다.
모든 개념, 모든 문장이 서로 연결되어
하나를 읽으면 전체가 움직입니다.
```

**결론**
```
올에서는 길을 잃지 않습니다.
당신이 선택한 올이 길이 됩니다.

깊이 파고들수록 높이 올라가고,
부분을 이해할수록 전체가 보입니다.

지혜의 올을 따라,
당신만의 천을 짜보세요.
```

### 6.3 브랜드 보이스 (Brand Voice)

#### 톤 (Tone)
- **명상적**: 서두르지 않고 깊이 생각하게 함
- **초대적**: 강요하지 않고 탐험을 권유함
- **지혜로운**: 가르치려 하지 않고 통찰을 나눔

#### 언어 스타일
- **간결함**: 짧은 문장, 명확한 의미
- **시적임**: 은유와 비유의 자연스러운 사용
- **존중함**: 사용자를 학습자이자 탐험가로 대우

#### 피해야 할 톤
- 교조적, 권위적
- 과도하게 기술적
- 상업적, 마케팅적

## 결론

**"올"은 단순함 속에 우주를 담은 이름입니다.**

한 음절, 두 글자에 담긴 의미:
- 실처럼 이어지는 연결망 (인드라망)
- 올 하나하나 따라가는 깊이 (Deep Dive)
- 천을 짜듯 엮는 인터랙티브 경험
- 오르다의 상승 (상구보리)
- 모두를 아우르는 전체성 (법계)

이 이름은 프로젝트의 기술적 아키텍처(Neural Network), 학습 철학(Deep Dive), 그리고 불교적 세계관(연기법, 화엄 사상)을 하나의 완벽한 은유로 통합합니다.

**"올"로 충분합니다.**
그 이상도, 그 이하도 아닙니다.
