## STORY

- [x] STORY 메인페이지와 book 세부페이지 분리
- [x] 태그체계 TEXT 태그 체계와 동일하게 수정.
- [x] STORY 콘텐츠 구조 정리 : series > part > chapter. 이 구조는 폴더구조인 동시에 메타데이터 위계구조. 메타데이터 수정.
- [x] STORY 메인페이지 : 기존 페이지 스타일 유지. 시리즈 카드 리스트 그리드 형식.
- [x] STORY 세부페이지 : STORY series 를 묶어서 보여주는 페이지로 STORY 시리즈 페이지를 이미함.
- [x] STORY 세부페이지 UI - 타이틀 섹션과 콘텐츠 섹션으로 구성. 콘텐츠 섹션은 탭구조 레이아웃 형식.
      탭구조 레이아웃 : 탭 해더 nav 버튼 - 서지정보, PART1, PART2, 등등.
      탭 서지정보 페이지 : 북커버, 서지정보, 다운로드버튼, 전체 책 목차, 내용 요약 설명 등,  라이선스.
      탭 본문영역 페이지(PART1, PART2 등등) : 왼쪽사이드바(해당 part 목차) + 오른쪽 본문.
- [x] STORY 세부 페이지 컨셉이미지 ![[STORY 세부 페이지 컨셉이미지1.png]]
- [x] 서지정보 카드스타일 박스로 수정
- [x] part 탭 본문에는 하나의 문서만 출력되게 수정
- [x] STORY, TEXT 본문 하단에 이전문서, 다음문서 버튼 추가
- [x] 시리즈 페이지 타이틀 하단 .ol-story-tag-row 태그들도 메인페이지 처럼 필터기능 추가하고 필터 결과로 관련 문서만 보여주는 기능 추가하기.
