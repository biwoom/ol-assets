## мқҙ л¬ём„ңлҘј мқҪкё° м „м—җ вҖ” м„Өкі„ кІ°м •мқҳ мқҙмң 

v1 м§ҖмӢңм„ңлҠ” "м–ҙл–»кІҢ л§Ңл“ңлҠ”к°Җ"лҘј лӢӨлӨҳлӢӨл©ҙ,  
v2лҠ” к·ё м „м—җ **"мҷң мқҙл ҮкІҢ м„Өкі„н•ҳлҠ”к°Җ"** лҘј лЁјм Җ м •лҰҪн•©лӢҲлӢӨ.

OLмқҖ лӢЁмҲң нҷҲнҺҳмқҙм§Җк°Җ м•„лӢҷлӢҲлӢӨ.

```

мһҘкё° лӘ©н‘ң: л¶Ҳкөҗ мҪҳн…җмё  м—°кё°л§қ (Dependent Origination Network)
```

мқҙ лӘ©н‘ңлҘј мІҳмқҢл¶Җн„° кө¬мЎ°м—җ л°ҳмҳҒн•ҳм§Җ м•Ҡмңјл©ҙ,  
мҪҳн…җмё к°Җ мҢ“мқјмҲҳлЎқ лӮҳмӨ‘м—җ кі м№ҳкё°к°Җ л§Өмҡ° м–ҙл ӨмӣҢм§‘лӢҲлӢӨ.  
**мҠӨнӮӨл§Ҳ м„Өкі„лҠ” мҪҳн…җмё ліҙлӢӨ лЁјм Җм—¬м•ј н•©лӢҲлӢӨ.**

# м„№м…ҳ A. м„Өкі„ мІ н•ҷ вҖ” мҷң м§ҖмӢқк·ёлһҳн”„мқёк°Җ

## A-1. нғңк·ё мӢңмҠӨн…ңмқҳ н•ңкі„

лҢҖл¶Җл¶„мқҳ мҪҳн…җмё  мӢңмҠӨн…ңмқҖ нғңк·ёлЎң кҙҖкі„лҘј н‘ңнҳ„н•©лӢҲлӢӨ.

```yaml
# мқјл°ҳм Ғмқё нғңк·ё л°©мӢқ
tags: [мҡ©мҲҳ, мӨ‘кҙҖн•ҷ, кіөм„ұ]
```

мқҙ л°©мӢқмқҳ л¬ём ңлҠ” **кҙҖкі„мқҳ мқҳлҜёк°Җ м—ҶлӢӨ**лҠ” кІғмһ…лӢҲлӢӨ.  
мҡ©мҲҳк°Җ мӨ‘кҙҖн•ҷмқ„ *м°ҪмӢңн–ҲлҠ”м§Җ*, *л№„нҢҗн–ҲлҠ”м§Җ*, *лӢЁмҲң м–ёкёүмқём§Җ* м•Ң мҲҳ м—ҶмҠөлӢҲлӢӨ.

OLмқҙ м§Җн–Ҙн•ҳлҠ” л°©мӢқ:

```yaml
# OL Ontology л°©мӢқ
relations:
  - subject: nagarjuna
    predicate: founded
    object: madhyamaka-school
  - subject: nagarjuna
    predicate: authored
    object: mulamadhyamakakarika
```

мЈјм–ҙ-кҙҖкі„-лӘ©м Ғм–ҙ **(Triple кө¬мЎ°)**к°Җ мқҳлҜёлҘј ліҙмЎҙн•©лӢҲлӢӨ.  
мқҙкІғмқҙ м§ҖмӢқк·ёлһҳн”„мқҳ н•өмӢ¬ лӢЁмң„мһ…лӢҲлӢӨ.

## A-3. л¶Ҳкөҗ мІ н•ҷкіјмқҳ м—°кІ°

м—°кё°лІ•(з·Јиө·жі•): лӘЁл“  кІғмқҖ мЎ°кұҙм—җ мқҳн•ҙ л°ңмғқн•ҳкі  мғҒнҳё м—°кІ°лҗҳм–ҙ мһҲлӢӨ.

```
л¶“лӢӨ вҶҗв”Җв”Җв”Җв”Җ к°ҖлҘҙміӨлӢӨ в”Җв”Җв”ҖвҶ’ мӮ¬м„ұм ң
мӮ¬м„ұм ң вҶҗв”Җв”Җ нҸ¬н•Ён•ңлӢӨ в”Җв”Җв”ҖвҶ’ нҢ”м •лҸ„
нҢ”м •лҸ„ вҶҗв”Җв”Җ нҸ¬н•Ён•ңлӢӨ в”Җв”Җв”ҖвҶ’ м •кІ¬
м •кІ¬ вҶҗв”Җв”Җв”Җв”Җ к·јкұ°н•ңлӢӨ в”Җв”Җв”ҖвҶ’ м—°кё°лІ•
м—°кё°лІ• вҶҗв”Җв”Җ м„ӨлӘ…н•ңлӢӨ в”Җв”Җв”ҖвҶ’ кіөм„ұ
кіөм„ұ вҶҗв”Җв”Җв”Җв”Җ л°ңм „мӢңмј°лӢӨ в”Җв”Җ мҡ©мҲҳ
мҡ©мҲҳ вҶҗв”Җв”Җв”Җв”Җ м ҖмҲ н–ҲлӢӨ в”Җв”Җв”ҖвҶ’ мӨ‘лЎ 
```

мқҙкІғмқҙ OLмқҙ л§Ңл“Өл ӨлҠ” **л¶Ҳкөҗ мҪҳн…җмё  м—°кё°л§қ**мқҳ лӘЁмҠөмһ…лӢҲлӢӨ.

## A-4. мӨ‘мҡ”н•ң кІҪкі  вҖ” кіјмһү м„Өкі„ кёҲм§Җ

мқҙ л¬ём„ңк°Җ м ңм•Ҳн•ҳлҠ” мҳЁнҶЁлЎңм§Җ кө¬мЎ°лҠ” **м җм§„м ҒмңјлЎң** мұ„мӣҢк°ҖлҠ” кІғмһ…лӢҲлӢӨ.

```
Phase 1: кө¬мЎ°л§Ң м„Өкі„, лҚ°мқҙн„°лҠ” л№„м–ҙмһҲм–ҙлҸ„ лҗЁ

```

мІҳмқҢл¶Җн„° мҲҳл°ұ к°ңмқҳ entityлҘј л§Ңл“Өл Ө н•ҳм§Җ л§Ҳм„ёмҡ”.  
**кө¬мЎ°к°Җ мҳ¬л°”лҘҙл©ҙ лҚ°мқҙн„°лҠ” мІңмІңнһҲ мұ„мҡё мҲҳ мһҲмҠөлӢҲлӢӨ.**

## C-2. Relation Type м •мқҳ

OL мҳЁнҶЁлЎңм§Җмқҳ н•өмӢ¬ кҙҖкі„ мң нҳ•мһ…лӢҲлӢӨ. лӢЁл°©н–Ҙ(вҶ’)мңјлЎң м •мқҳн•©лӢҲлӢӨ.

### мқёл¬ј кҙҖл Ё кҙҖкі„
```
person вҶ’ text       : authored (м ҖмҲ н–ҲлӢӨ)
person вҶ’ concept    : developed (л°ңм „мӢңмј°лӢӨ), taught (к°ҖлҘҙміӨлӢӨ)
person вҶ’ person     : student_of (м ңмһҗмқҙлӢӨ), teacher_of (мҠӨмҠ№мқҙлӢӨ)
person вҶ’ place      : born_in (нғңм–ҙлӮ¬лӢӨ), lived_in (лЁёл¬јл ҖлӢӨ), died_in (мһ…м Ғн–ҲлӢӨ)
person вҶ’ school     : founded (м°ҪмӢңн–ҲлӢӨ), belonged_to (мҶҚн–ҲлӢӨ)
person вҶ’ event      : participated_in (м°ём—¬н–ҲлӢӨ), experienced (кІҪн—ҳн–ҲлӢӨ)
```

### л¬ён—Ң кҙҖл Ё кҙҖкі„
```
text вҶ’ concept      : explains (м„ӨлӘ…н•ңлӢӨ), discusses (л…јн•ңлӢӨ)
text вҶ’ text         : commentary_on (мЈјм„қмқҙлӢӨ), quotes (мқёмҡ©н•ңлӢӨ), responds_to (л°ҳлЎ н•ңлӢӨ)
text вҶ’ person       : mentions (м–ёкёүн•ңлӢӨ)
text вҶ’ school       : represents (лҢҖн‘ңн•ңлӢӨ)
```

### к°ңл…җ кҙҖл Ё кҙҖкі„
```
concept вҶ’ concept   : related_to (кҙҖл ЁлҗңлӢӨ), includes (нҸ¬н•Ён•ңлӢӨ),
                      contrasts_with (лҢҖл№„лҗңлӢӨ), derived_from (мң лһҳн•ңлӢӨ)
concept вҶ’ practice  : realized_through (мӢӨнҳ„лҗңлӢӨ)
concept вҶ’ school    : central_to (н•өмӢ¬мқҙлӢӨ)
```

### н•ҷнҢҢ/м „нҶө кҙҖл Ё кҙҖкі„
```
school вҶ’ school     : emerged_from (нҢҢмғқлҗҗлӢӨ), influenced (мҳҒн–ҘмӨ¬лӢӨ)
school вҶ’ text       : canonical_text (мҶҢмқҳкІҪм „мқҙлӢӨ)
school вҶ’ place      : originated_in (л°ңмӣҗн–ҲлӢӨ)
```

## E-1. `src/lib/relations.ts` вҖ” кҙҖкі„ лҚ°мқҙн„° мІҳлҰ¬

```typescript
import { getCollection } from 'astro:content';

export type Triple = {
  subject: string;
  predicate: string;
  object: string;
  source?: string;
  note?: string;
};

/**
 * лӘЁл“  collectionм—җм„ң tripleмқ„ мҲҳм§‘н•ҳм—¬ м§ҖмӢқк·ёлһҳн”„ лҚ°мқҙн„° л°ҳнҷҳ
 * build мӢңм—җл§Ң мӢӨн–үлҗЁ (Astro SSG)
 */
export async function buildKnowledgeGraph() {
  const [books, entities, designs, blogs] = await Promise.all([
    getCollection('book'),
    getCollection('entities'),
    getCollection('design'),
    getCollection('blog'),
  ]);

  const triples: Triple[] = [];
  const entityMap = new Map<string, any>();

  // Entity л…ёл“ң мҲҳм§‘
  for (const entity of entities) {
    entityMap.set(entity.data.id, entity);
    // entity мһҗмІҙмқҳ relations мҲҳм§‘
    for (const rel of entity.data.relations ?? []) {
      triples.push(rel);
    }
  }

  // Book л¬ём„ңмқҳ relations мҲҳм§‘
  for (const book of books) {
    for (const rel of book.data.relations ?? []) {
      triples.push({ ...rel, source: book.slug });
    }
  }

  return { triples, entityMap };
}

/**
 * нҠ№м • entityмқҳ лӘЁл“  кҙҖкі„ л°ҳнҷҳ (м–‘л°©н–Ҙ)
 */
export async function getEntityRelations(entityId: string) {
  const { triples } = await buildKnowledgeGraph();

  const asSubject = triples.filter(t => t.subject === entityId);
  const asObject  = triples.filter(t => t.object  === entityId);

  return { outgoing: asSubject, incoming: asObject };
}

/**
 * нҠ№м • entityлҘј м–ёкёүн•ҳлҠ” лӘЁл“  л¬ём„ң л°ҳнҷҳ (backlinks)
 */
export async function getBacklinks(entityId: string) {
  const books   = await getCollection('book');
  const designs = await getCollection('design');
  const blogs   = await getCollection('blog');

  const allDocs = [...books, ...designs, ...blogs];

  return allDocs.filter(doc =>
    (doc.data.entities ?? []).includes(entityId)
  );
}
```

## I-3. relation мҲңнҷҳ м°ёмЎ° мЈјмқҳ

```

```

мҲңнҷҳ мһҗмІҙлҠ” л¶Ҳкөҗ м—°кё°лЎ м—җм„ң мһҗм—°мҠӨлҹҪмҠөлӢҲлӢӨ.  
лӢЁ, л ҢлҚ”л§Ғ мӢң л¬ҙн•ң лЈЁн”„лҘј л°©м§Җн•ҳлҠ” к№Ҡмқҙ м ңн•ңмқҙ н•„мҡ”н•©лӢҲлӢӨ.

```typescript
// lib/relations.tsм—җ м¶”к°Җ
export function traverseRelations(
  entityId: string,
  triples: Triple[],
  maxDepth: number = 2,  // к№Ҡмқҙ м ңн•ң
  visited = new Set<string>()
): Triple[] {
  if (visited.has(entityId) || maxDepth === 0) return [];
  visited.add(entityId);

  const direct = triples.filter(t => t.subject === entityId);
  const nested = direct.flatMap(t =>
    traverseRelations(t.object, triples, maxDepth - 1, visited)
  );

  return [...direct, ...nested];
}
```
