# DESIGN 스타일시트 & 레퍼런스 제작 매뉴얼

> **대상**: 불교 콘텐츠 삽화·웹툰·애니메이션 제작을 위한 디자인 레퍼런스를 만들려는 사람
> **목적**: 업계 표준 캐릭터 디자인 시트 제작법을 OL 불교 콘텐츠에 적용하는 방법을 안내
> **기준일**: 2026-06-01

---

## 0. 디자인 레퍼런스란

디자인 레퍼런스(Design Reference)는 캐릭터, 배경, 소품 등의 **시각적 설계도**입니다. 애니메이션·게임·웹툰 업계에서는 이것을 기반으로 여러 작업자가 동일한 대상을 일관되게 그립니다.

OL DESIGN은 불교 콘텐츠에 등장하는 인물·건축물·소품의 레퍼런스를 CC0로 공개하여, 누구든 불교 삽화·웹툰·애니메이션 제작에 활용할 수 있도록 합니다.

### 레퍼런스의 종류

```
캐릭터 레퍼런스     인물의 외형·의상·표정·포즈를 정의
배경 레퍼런스       장소·건축물·자연환경의 구조와 분위기를 정의
소품 레퍼런스       도구·의례 용품·일상 물건의 형태와 비례를 정의
```

## 1. 캐릭터 디자인 시트 — 구성 요소

애니메이션·게임 업계에서 캐릭터 디자인 시트는 보통 다음 시트들로 구성됩니다.
하나의 캐릭터에 대해 여러 장의 시트를 만들고, 각 시트가 서로 다른 정보를 담당합니다.

### 1.1 시트 구성 전체 목록

```
번호  시트명                     영문명                   필수도
─────────────────────────────────────────────────────────────
 1   턴어라운드 시트             Turnaround Sheet         필수
 2   표정 시트                  Expression Sheet         필수
 3   의상·소품 시트              Costume & Props Sheet    필수
 4   비례·신체 구조도            Proportion Chart         권장
 5   컬러 시트                  Color Model Sheet        권장
 6   액션·포즈 시트              Action/Pose Sheet        선택
 7   나이별 변화 시트            Age Progression Sheet    선택
 8   비교 시트                  Lineup/Comparison Sheet  선택
```

### 1.2 각 시트의 역할

**1. 턴어라운드 시트 (Turnaround Sheet)** — 가장 핵심

캐릭터를 동일 포즈로 **정면·3/4면·측면·후면** 네 방향에서 그린 시트입니다.
3D 모델링, 애니메이션 원화, 웹툰 작화 모두 이 시트를 기본으로 참조합니다.

```
구성 요소:
  - 4~8방향 전신 뷰 (정면, 3/4 정면, 측면, 3/4 후면, 후면)
  - 모든 뷰가 동일 수평선(바닥선) 위에 정렬
  - 머리 높이를 기준으로 등신 눈금선 표시
  - 차분한 기본 자세 (A-pose 또는 T-pose)
  - 배경 없이 단색 또는 투명 배경
```

**2. 표정 시트 (Expression Sheet)**

캐릭터의 감정 표현을 모아놓은 시트입니다.

```
구성 요소:
  - 기본 표정 (무표정/평온)
  - 6대 기본 감정: 기쁨, 슬픔, 분노, 놀람, 두려움, 혐오
  - 캐릭터 고유 표정: 선정(禪定) 중 미소, 설법 시 자비로운 표정 등
  - 얼굴 3/4 앵글 통일, 동일 크기로 배치
  - 각 표정 아래에 감정명 라벨 표기
```

**3. 의상·소품 시트 (Costume & Props Sheet)**

캐릭터가 착용하는 의복과 소지품을 분해하여 보여줍니다.

```
구성 요소:
  - 의상 각 부위 분리도 (상의, 하의, 외투, 장신구 등)
  - 착용 순서 (레이어링 다이어그램)
  - 옷감 재질·패턴 표현 (면, 삼베, 비단 등)
  - 소품 개별 확대도 (발우, 석장, 염주 등)
  - 색상 지정 (컬러 코드 또는 팔레트)
```

**4. 비례·신체 구조도 (Proportion Chart)**

등신 비율과 주요 신체 비례를 수치로 기록합니다.

```
구성 요소:
  - 등신 비율 (예: 7등신, 8등신)
  - 머리:어깨:허리:엉덩이 폭 비율
  - 팔·다리 길이 비율
  - 체형 특징 메모 (마른 체형, 근육질 등)
  - 다른 캐릭터와의 키 비교선 (선택)
```

**5. 컬러 시트 (Color Model Sheet)**

최종 채색 상태를 정의합니다.

```
구성 요소:
  - 기본 컬러 (주광 조건)
  - 그림자 컬러
  - 하이라이트 컬러
  - 피부·머리카락·의상 각 부위별 컬러 코드 (HEX 또는 RGB)
  - 야간/실내 등 조명 변형 (선택)
```

## 2. 실전 예시 — 아난다 존자 캐릭터 디자인

아난다(Ananda)는 부처님의 시자(侍者)이자 다문제일(多聞第一)의 제자입니다.
이 예시는 아난다 존자의 캐릭터 디자인 시트를 어떻게 기획하고 제작하는지 보여줍니다.

### 2.1 사전 조사 — 캐릭터 설정 정리

디자인을 시작하기 전에 문헌 기반의 캐릭터 설정을 정리합니다.

```
캐릭터명:    아난다 (Ananda, 阿難陀)
역할:        부처님의 시자(수행시중), 다문제일
시대:        기원전 6~5세기
지역:        고대 인도 마가다국, 코살라국 일대
나이:        부처님보다 약간 젊음, 출가 시 20대 초반
체형:        보통~마른 체형, 단정한 인상
성격:        온화, 섬세, 기억력 우수, 겸손

의상:
  - 삼의(三衣): 승가리(상의), 울다라승(중의), 안타회(내의)
  - 기본색: 괴색(壞色) — 황토·적갈색 계열 (화려한 색 금지)
  - 맨발
  - 발우(鉢盂)를 항상 휴대

외형 특징 (경전 근거):
  - 단정한 용모 (경전에 "용모단정"이라 기록됨)
  - 삭발 (출가 수행자)
  - 이마에 백호 없음 (백호는 부처님만의 상호)
  - 귀가 크게 늘어지지 않음 (부처님의 대이상과 구별)

참고 경전:
  - 증일아함경, 장아함경
  - 대반열반경 (부처님 입멸 장면)
  - 아난다 관련 설화 다수
```

## 3. 배경 레퍼런스 — 건축물·장소

불교 콘텐츠의 배경은 크게 **사찰·정사**, **자연환경**, **도시·마을**로 나뉩니다.
배경 레퍼런스는 캐릭터와 달리 **공간의 구조와 분위기**를 전달하는 것이 목표입니다.

### 3.1 배경 레퍼런스 시트 구성

```
번호  시트명                    영문명                    필수도
──────────────────────────────────────────────────────────
 1   외관 전경도               Exterior Overview         필수
 2   평면도·배치도              Floor Plan / Layout       필수
 3   주요 구조물 상세도          Structure Detail          권장
 4   단면도                    Cross Section             권장
 5   분위기·조명 시트           Mood / Lighting Sheet     선택
 6   시간대별 변화              Time-of-Day Variation     선택
```

### 3.2 실전 예시 — 기원정사(祇園精舍, Jetavana)

기원정사는 부처님이 가장 오래 머문 정사로, 불교 경전 다수의 배경입니다.

## 4. 소품 레퍼런스

소품은 캐릭터가 사용하거나 장면에 등장하는 물건입니다.
하나의 시트에 관련 소품을 모아 **다각도 + 실척 비례**로 정리합니다.

### 4.1 소품 레퍼런스 시트 구성

```
번호  시트명                   영문명                필수도
──────────────────────────────────────────────────────
 1   다각도 뷰                Multi-angle View      필수
 2   비례·치수 도면            Scale Drawing         필수
 3   재질·색상 표기            Material & Color      권장
 4   사용 장면 스케치           Usage Context         선택
```

### 4.2 실전 예시 — 비구의 여섯 가지 소지품(六物)

초기불교 비구가 허용된 소지품은 삼의(三衣) + 발우 + 좌구 + 여과기입니다.

## 5. 제작 가이드라인

### 5.1 파일 규격

```
해상도:     최소 3000px (긴 변 기준), 권장 4000px
포맷:       PNG (선화·투명 배경) 또는 JPG (채색 완성)
            WebP로 추가 변환하면 용량 절감 가능
색상 모드:  sRGB
비트 깊이:  8bit (웹용)
파일 크기:  장당 2MB 이하 권장
```

### 5.2 시트 레이아웃 원칙

```
1. 여백:       시트 가장자리에 최소 5%의 여백
2. 정렬:       모든 뷰는 동일 바닥선 위에 정렬
3. 눈금선:     등신·비례를 보여주는 수평선 표시
4. 라벨:       각 뷰·부위에 한글 라벨 표기
5. 배경:       단색 (밝은 회색 #F0F0F0 또는 #E8E8E8)
6. 캔버스:     가로 배치를 기본으로 (가로 > 세로)
```

### 5.3 불교 콘텐츠 고유 주의사항

```
의상:
  - 괴색(壞色) 원칙: 승복은 화려한 원색 금지, 황토·적갈·회갈색 계열
  - 편단우견(偏袒右肩): 오른쪽 어깨를 드러내는 착용법이 기본
  - 통견(通肩): 양쪽 어깨를 모두 덮는 착용법은 특정 상황에서만 사용
  - 시대·지역에 따라 승복 형태가 다름에 주의

신체:
  - 부처님(삼십이상): 백호, 육계, 대이상 등 특수 표현이 있으나
    제자·일반 인물에게는 적용하지 않음
  - 삭발 표현: 완전 삭발 또는 극히 짧은 머리, 면도 자국은 선택

소품:
  - 발우(鉢)는 초기불교 시대 철제 또는 도기, 금·은 소재 금지
  - 석장(錫杖)은 중국·한국·일본 불교 전통, 초기불교에서는 사용하지 않음
  - 염주도 후대 전통, 초기불교 레퍼런스에서는 생략

건축:
  - 초기불교 시대 건축은 소박한 목조+초가 구조
  - 화려한 기와·단청은 동아시아 불교 전통
  - 시대를 명시하여 혼동 방지
```

### 5.4 시리즈 일관성

같은 시리즈의 레퍼런스는 아래 요소를 통일해야 합니다.

```
  - 등신 비율 (같은 작품 내 모든 캐릭터 동일 기준)
  - 선 굵기와 스타일
  - 컬러 팔레트 톤 (채도·명도 범위)
  - 라벨·주석 폰트와 크기
  - 캔버스 크기와 여백
```

```

홍보·전시용으로 디자인한 것이다              → poster

자연환경·지역의 모습이다                     → landscape

유물·도구·소품의 형태를 보여준다             → artifact
위 어디에도 해당하지 않는다                  → other
```

## 7. 제작 워크플로우 요약

하나의 캐릭터(예: 아난다 존자)에 대한 전체 작업 흐름입니다.

```
1. 사전 조사
   └─ 경전·문헌에서 외형·성격·소지품 정보 수집
   └─ 시대·지역 고증 자료 확인

2. 설정 문서 작성
   └─ 캐릭터 설정표 (2.1절 참조)
   └─ 컬러 팔레트 초안

3. 러프 스케치
   └─ 턴어라운드 러프 (비례·실루엣 확정)
   └─ 표정 러프
   └─ 의상 분해 러프

4. 클린업 (선화 정리)
   └─ 동일 선 굵기·스타일로 통일
   └─ 눈금선·라벨 배치

5. 채색
   └─ 기본색 → 그림자 → 하이라이트
   └─ 컬러 코드 추출·기록

6. 시트 조합
   └─ 각 시트를 최종 캔버스에 배치
   └─ 라벨·주석 최종 확인

7. OL DESIGN 등록
   └─ 이미지 → public/design/
   └─ 메타데이터 md → src/content/design/
   └─ published: true 설정
   └─ 빌드 확인
```
