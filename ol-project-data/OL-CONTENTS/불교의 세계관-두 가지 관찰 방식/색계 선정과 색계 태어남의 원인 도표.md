# 색계 선정과 색계 태어남의 원인 도표

| 색계 단계      | 태어나는 세계                                                            | 원인이 되는 선정 | 주요 마음요소     | 산스크리트                                         | 설명                                                |
| ---------- | ------------------------------------------------------------------ | --------- | ----------- | --------------------------------------------- | ------------------------------------------------- |
| **초선初禪**   | 범중천梵衆天, 범보천梵輔天, 대범천大梵天                                             | 초선정初禪定    | 심尋·사伺·희喜·락樂 | **vitarka**, **vicāra**, **prīti**, **sukha** | 감각욕을 떠났지만 아직 거친 사유 작용인 심·사와 선정에서 생긴 기쁨·즐거움이 함께 있음 |
| **제2선第二禪** | 소광천少光天, 무량광천無量光天, 극광정천極光淨天                                         | 제2선정第二禪定  | 희喜·락樂       | **prīti**, **sukha**                          | 심尋·사伺가 사라지고, 내적 고요 속에서 선정의 기쁨과 즐거움이 강하게 나타남       |
| **제3선第三禪** | 소정천少淨天, 무량정천無量淨天, 변정천遍淨天                                           | 제3선정第三禪定  | 락樂·평정捨      | **sukha**, **upekṣā**                         | 희喜의 들뜸이 가라앉고, 깊은 즐거움과 평정이 중심이 됨                   |
| **제4선第四禪** | 무운천無雲天, 복생천福生天, 광과천廣果天, 무번천無煩天, 무열천無熱天, 선현천善現天, 선견천善見天, 색구경천色究竟天 등 | 제4선정第四禪定  | 평정捨         | **upekṣā**                                    | 락樂마저 초월하고 괴로움도 즐거움도 아닌 평정과 청정한 알아차림이 중심이 됨        |

# 마음요소별 정리

| 한역            | 산스크리트       | 의미                               | 색계 선정에서의 위치             |
| ------------- | ----------- | -------------------------------- | ----------------------- |
| **심尋**        | **vitarka** | 대상을 향해 마음을 처음으로 향하게 하는 작용, 거친 사유 | 초선에만 있음                 |
| **사伺**        | **vicāra**  | 대상에 머물며 살피는 작용, 미세한 고찰           | 초선에만 있음                 |
| **희喜**        | **prīti**   | 선정에서 생기는 기쁨, 환희, 고양감             | 초선·제2선에 있음              |
| **락樂**        | **sukha**   | 선정에서 생기는 안락, 편안한 즐거움             | 초선·제2선·제3선에 있음          |
| **평정平靜 / 사捨** | **upekṣā**  | 치우침 없는 고요함, 평등한 마음               | 제3선에서 두드러지고 제4선에서 중심이 됨 |

# 핵심 흐름

| 선정의 심화    | 사라지는 요소 | 남는 요소   | 태어나는 색계 |
| --------- | ------- | ------- | ------- |
| 욕계 → 초선   | 감각욕     | 심·사·희·락 | 초선천     |
| 초선 → 제2선  | 심·사     | 희·락     | 제2선천    |
| 제2선 → 제3선 | 희       | 락·평정    | 제3선천    |
| 제3선 → 제4선 | 락       | 평정      | 제4선천    |

---

따라서 색계에 태어나는 원인을 아주 압축하면 다음과 같습니다.

> **색계에 태어나는 원인은 색계 선정과 결합한 유루 선업이다.
> 그 선정은 초선에서 제4선까지 점차 심尋·사伺·희喜·락樂이 소멸하고, 마지막에는 평정捨, 즉 upekṣā가 중심이 되는 방식으로 깊어진다.**

문서에 넣을 때는 “색계는 단순히 선행善行만으로 태어나는 세계가 아니라, 욕계의 감각욕을 떠난 선정업禪定業에 의해 태어나는 세계”라고 설명하면 교학적으로 안정적입니다.
